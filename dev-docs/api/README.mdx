# KPM API

KPM と連携したプラグインを作成するための API です。

## 概要

KPM APIは、 KPM と連携したプラグインを作成するための Java で呼び出し可能な API です。

## チュートリアル

### ステップ1. KPM 情報ファイルを記述する

KPM API の全ての機能を利用するためには、まず KPM 情報ファイルを記述する必要があります。  
KPM 情報ファイルは、プラグインについての注釈をKPM に与える機能です。プラグインの開発者が、自分のプラグインに埋め込んで使用します。   
これにより、依存関係の適切な解決を行ったリ、サーバに未対応なプラグインのロードを防げます。

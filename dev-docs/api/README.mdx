# KPM API

KPM と連携したプラグインを作成するための API です。

---

## 概要

KPM API は、 KPM と連携したプラグインを作成するための Java で呼び出し可能な API です。

それぞれの詳細な説明については、各リンク先を参照してください。

###  [KPM フック](/dev-docs/api/hook)  

<BelowDocument docId="api/hook" message="インストール時やアンインストール時に、あなたのプラグインの任意コードを実行します。" />

KPM API の全ての機能を利用するためには、まず KPM 情報ファイルを記述する必要があります。

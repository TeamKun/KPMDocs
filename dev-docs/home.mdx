# KPM 開発者ポータル

このドキュメントは、プラグイン開発者向けに提供されている API や、特別な機能を利用するための情報を提供しています。  
あなたが開発者でない場合や、 KPM を使用してプラグインをインストールする方法は、[こちら](/docs/use-kpm) を参照してください。

## チュートリアル

## KPM API について

KPM は、以下の機能を API として公開しています。これらの API を利用することで、KPM と連携したプラグインを作成できます。  
それぞれの詳細な説明については、各リンク先を参照してください。

+ [KPM 情報ファイル](/dev-docs/api-docs/kpm-info)  
  プラグインに KPM 用の注釈を記述して、KPM にプラグインの情報を提供します。
+ [KPM フック](/dev-docs/api-docs/hook)  
  インストール時やアンインストール時に、 プラグインのメソッドを呼び出します。

## プラグイン開発時の注意事項

# エクスポートコマンド

プラグインの設定ファイルとダウンロードリンクを含んだファイルを生成します。

---

<Deprecation version="v3" removeMode />

## コマンド概要 - <kbd>/kpm export</kbd>

プラグインの設定ファイルとダウンロードリンクを含んだファイルを生成します。  
生成されたファイルは、 [file.io](https://file.io) に自動でアップロードされます。このリンクは一度のみ有効で、 [<kbd>import</kbd>](#import) コマンドで使用できます。

:::caution

KPM がプラグインを解決できない場合、エクスポートから除外されます。

:::

### エイリアス

_\(なし)_

### コマンド構文

- `export <all|プラグイン名> [プラグイン名2…]`

第一引数に `all` を指定すると、サーバにインストールされているすべてのプラグインの設定ファイルとダウンロードリンクを含んだファイルを生成します。

### 権限

- `kpm.export`
- `kpm.*`

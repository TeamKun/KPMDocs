# プラグイン情報コマンド

プラグインの詳細な情報を表示します。

---

## コマンド概要 - <kbd>/kpm info</kbd>

プラグインの詳細な情報を表示します。

<details>
  <summary>表示される情報一覧</summary>

- 名前
- 作者
- 状態
- 読み込みタイミング
- 保護状態
- ウェブサイト
- 説明

- ファイル名
- ファイルサイズ

- 依存関係
- 非依存関係

- コマンド一覧  
  コマンド一覧 をホバーすると、そのコマンドの詳細説明が表示されます。

</details>

### エイリアス

_\(なし)_

### コマンド構文

- `info <プラグイン名>`

### 権限

- `kpm.info`
- `kpm.*`

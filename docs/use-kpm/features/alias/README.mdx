# エイリアス

プラグインに別名を付けてかんたんにインストールします。

---

## 概要

エイリアスは、リモートによって管理されるプラグインに別名をつけられる機能です。これにより、 `ExamplePlugin` を、 `ex` のような短縮された別前でインストールできます。

<Since version={"v3"} />

---

## エイリアスを使う

エイリアスを用いたインストールは、通常のインストールと同じように、クエリを指定する場所にエイリアス名を指定します。

:::tip

これらは、[AliasResolver](../resolver#aliasresolver) によって解決されます。

:::

# コマンドの書き記し方

このドキュメントにおける、コマンド構文の表し方について学びます。

---

## 一般的な構文記述

このドキュメントでは、コマンドを以下のように記述しています：

```
/command <required argument> [optional argument]
```

`<required argument>` は必須の引数を表し、`[optional argument]` はオプションの引数を表します。  
引数とは、コマンドに与える値のことです。一般に、コマンド名の後に空白を入れて実行します。

## 可変長引数における構文記述

また、以下のような記述も見られるでしょう：

```
/command <required argument> [optional argument...]
```

`...` がある引数は、オプションの引数を複数指定できることを表します（可変長引数）。  
この場合、 `[optional argument...]` は、０個以上、無限個のオプションの引数を表します。
